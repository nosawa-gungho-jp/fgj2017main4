＜概要＞

このアセットはサウンド周りのちょっと気の利いた機能を提供します。
・BGM、SEといったカテゴリー単位でのボリューム調整
・ミュート機能
・ボリュームフェード処理
・パン、ピッチ処理
・ポーズ処理
・サウンドデータ管理
